# Stone记 (Stone Ji) 视频内容全文检索与文字稿导出工具



## 📝 项目简介
这是一个专门为 **Stone记** 观众和研究者开发的文字稿检索平台。通过技术手段将数千期视频内容转化为可搜索的文本数据库，解决“视频信息难以二次检索”的痛点。

### 🌐 访问地址
**官方网站：[https://www.csmx.cc](https://www.csmx.cc)**

---

## ✨ 核心功能
* **全文检索：** 支持搜索 Stone记 历年视频中的任何关键词。
* **秒级定位：** 搜索结果直接对应视频时间戳，点击即可跳转。
* **文字稿导出：** 方便引用和深度研究。
* **持续更新：** 每日同步更新最新一期视频内容。

## 🚀 为什么使用本平台？
Stone记的会员增长遇到天花板，很大程度上是因为内容沉淀在视频中无法被有效利用。本平台旨在通过 SEO 爬虫技术和文本化处理，释放时政自媒体的长尾价值。
